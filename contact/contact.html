Tournament maker

is designed by Sjoerd Hemels and Wiljan Vinke during their training at YER IT.
This project is designed to show understanding of a few key principles:
  • Object oriented programming in Java 8
  • Database design with MySQL
  • Understanding and using REST
  • Webdevelopment (HTML, CSS and Javascript)
  • Test-driven development with JUnit
  • Building tool Maven
  • Auto-deployment with Jenkins
